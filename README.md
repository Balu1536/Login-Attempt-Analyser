# 🔐 Login Attempt Analyzer

A cybersecurity-focused data analysis project that analyzes login activity records and identifies suspicious login attempts using Python, Pandas, and Matplotlib.

## 📌 Project Overview

The Login Attempt Analyzer processes login event data to detect failed logins, attack attempts, and account takeover incidents. It generates insightful visualizations and a dashboard for monitoring user activity and security trends.

## 🚀 Features

* Analyze login records from CSV files
* Count successful and failed login attempts
* Detect suspicious IP addresses
* Identify users with repeated failed logins
* Perform country-wise and device-wise analysis
* Browser and operating system analysis
* Account takeover monitoring
* Interactive cybersecurity dashboard with KPI cards and charts

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
